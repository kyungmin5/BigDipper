# ⭐ 북두칠성 독서 앱

## 🖥️ 프로젝트 소개
2023-1학기에 참여한 모바일 프로그래밍 수업 과제의 일부로 진행한 프로젝트입니다. <br>
4명의 팀원과 협업하여 제작하였으며, 북클럽 및 독서 앱을 구현하였습니다.

## 🕰️ 개발 기간
2023.05.12 ~ 2023.06.22
